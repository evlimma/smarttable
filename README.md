# SmartTable @evlimma

###### Make creating HTML tables easy with PHP

Facilite a criação de tabelas HTML com PHP


### Highlights

- Easy to set up (Fácil de configurar)


## Installation

Data Layer is available via Composer:

```bash
"evlimma/smarttable": "1.0.*"
```

or run

```bash
composer require evlimma/smarttable
```
